# ruby-lsp

Ruby language server for Claude Code, providing code intelligence and analysis.

## Supported Extensions
`.rb`, `.rake`, `.gemspec`, `.ru`, `.erb`

## Installation

### Via gem (recommended)
```bash
gem install ruby-lsp
```

### Via Bundler
Add to your Gemfile:
```ruby
gem 'ruby-lsp', group: :development
```

Then run:
```bash
bundle install
```

## Requirements
- Ruby 3.0 or later
